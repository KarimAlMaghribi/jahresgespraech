# Jahresgespräch Frontend

Eine kleine Single-Page-Anwendung auf Basis von Vite, React (TypeScript) und Tailwind CSS. Die Seite sammelt die wichtigsten Punkte
für das Jahresgespräch (Rückblick, Business-Impact, Wachstum, Engagement und Ausblick) und bietet eine druckoptimierte Ansicht.

## Setup

1. **Abhängigkeiten installieren**
   ```bash
   npm install
   ```

2. **Entwicklungsserver starten**
   ```bash
   npm run dev
   ```
   Der Server läuft standardmäßig unter http://localhost:5173.

3. **Lints ausführen**
   ```bash
   npm run lint
   ```

4. **Produktion bauen**
   ```bash
   npm run build
   ```

5. **Produktionsergebnis lokal ansehen**
   ```bash
   npm run preview
   ```

> Hinweis: Falls Paket-Downloads aufgrund von Netzwerk- oder Proxy-Einstellungen scheitern, müssen diese vor dem ersten `npm install`
> angepasst werden.
